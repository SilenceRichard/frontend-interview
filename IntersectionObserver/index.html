Intersection Observer API 演示

Intersection Observer API 是一个现代浏览器提供的强大 API，它能够异步监测目标元素与其祖先元素或视口之间的交叉状态变化。以下是三个常见应用场景的演示：

📷

图片懒加载

演示如何使用 Intersection Observer 实现图片懒加载，仅在图片即将进入视口时才加载，大幅优化页面性能和带宽使用。

♾️

无限滚动加载

演示如何实现无限滚动，当用户快要滚动到页面底部时，自动触发加载更多内容，提高浏览体验。

▶️

视频自动播放/暂停

演示如何根据视频在视口中的可见状态，自动控制视频的播放和暂停，优化用户体验与性能。
